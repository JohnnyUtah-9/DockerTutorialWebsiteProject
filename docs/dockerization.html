Docker: What is Docker and 'Dockerization'?

Docker is a platform that gives users the capability to to package and run applications inside an isolated environment reffered to as a 'container.' Docker's website provides useful information on docker uses and applications such as the following:

  1. Docker runs off the central CPU kernel and therfore utilizes less resources than virtual machines
  2. Docker allows you to develop your application and its supporting components using containers.
  3. The container becomes the unit for distributing and testing your application.

What is ICANN?

ICANN, stands for the Internet Corporation for Assigned Names and Numbers(ICANN). According to a Princeton University research paper entitled: INTERNATIONAL REGIMES OF THE INTERNET AND AVIATION: STRUCTURE, PREFERENCES, AND TECHNOLOGY, ICANN "holds control over the domain name system (DNS), responsible for managing the addressing system of the Internet.
