What is PyCharm?

PyCharm: is an integrated development environment, developed by JetBrains. The IDE is cross-platform and can be installed on Windows, Linux or MAC. It provides code analysis, a graphical debugger, an integrated unit tester, and integration with version control.

What does PyCharm Provide?

• Coding assistance and analysis, with code completion, syntax and error highlighting, linter integration, and quick fixes

• Project and code navigation: specialized project views, file structure views and quick jumping between files, classes, methods and usages

• Python refactoring: includes rename, extract method, introduce variable, introduce constant, pull up, push down and others

• Support for web frameworks: Django, web2py and Flask [professional edition only]

• Integrated Python debugger

• Integrated unit testing, with line-by-line code coverage

• Google App Engine Python development [professional edition only]

• Version control integration: unified user interface for Mercurial, Git, Subversion, Perforce and CVS with change lists and merge

• Support for scientific tools like matplotlib, numpy and scipy [professional edition only]

Tutorial

Go to your desktop and run PyCharm

1. Click on File ---> Settings ---> Build, Execution, Deployment ---> Docker

2. Select the docker and add the docker container inside the window using + icon

3. Once successful, now we can create the Dockerfile

4. Type the following code in the Dockerfile:

- FROM python:3

- ADD src/src

- RUN pip install pystrich

- CMD [ "python", "./src/myscript.py" ]

5. Commit

6. Create the myscript.py As it has to be referenced to the file we entered inside Dockerfile.

7. Click on the + and add the Dockerfile, then make sure the following is configured:

- Dockerfile: Dockerfile

- imagetag: pythongprogramimage

- container name:firstpythoncontainer

- Select Run Build Image

- Click Apply and Ok

8. Run the Dockerfile

9. Add the Interpreter,select Docker, and make sure the image name is: pythonprogramimagee
