What are the Differences between containerization and virtuaization?

Containierzation

Containerization is a less resource intensive alternative to machine virtualization that involves encapsulating an application in a container from inside its own operating environment. Containers use the same host operating system (OS) instead of installing an OS for each guest VM and Containers also provide a way to isolate applications and provide a virtual platform for applications to run on.
