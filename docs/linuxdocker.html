Tutorial

In this tutorial we will demonstrate how to use Docker with some linux commands through the command line.

Getting Started: First we must pull the linux image inside the command line.

1. Command: docker pull ubuntu - This will pull the ubuntu image.

2. Use the "Docker Images" to check the existing Docker Images on your computer.

3. Command: docker run -it ubuntu:latest /bin/bash - this will run Ubuntu to start a docker container, and logs in the Ubuntu container.

4. Command: apt-get update - this command will allow you to install applications in the linux container.

5. Command: apt-get install apache2 - this command will allow you to install an apache server within the container.

6. Command: exit - this command closes the container.
